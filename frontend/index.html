About This Project

This project was developed to simplify the analysis of Electron Energy Loss Spectroscopy (EELS) data, a task that can be both complex and time-consuming. As a Physics PhD student, I encountered these challenges firsthand, which inspired the creation of this tool. The goal is to make EELS data analysis faster and more accessible, allowing researchers to focus more on their scientific discoveries rather than the intricacies of data processing.
